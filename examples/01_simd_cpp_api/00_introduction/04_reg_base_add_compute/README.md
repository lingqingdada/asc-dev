# RegBase Add样例介绍

## 概述

本样例介绍了基于Ascend C的Add算子核函数直调方法（RegBase场景），实现两个输入张量的逐元素相加，展示了片上存储和寄存器层级的向量计算流程。

## 算子开发样例

|  目录名称                                                   |  功能描述                                              |
| ------------------------------------------------------------ | ---------------------------------------------------- |
| [vector_add](./vector_add) | 本样例演示了SIMD场景下，基于RegBase编程范式的Add算子的核函数直调方法，算子支持单核运行，不同流水线之间使用VEC_LOAD和VEC_TORE同步 |
