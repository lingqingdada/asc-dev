# MindStudio工具样例介绍

## 概述

本样例基于MindStudio工具，演示了msProf和msSanitizer两个调测工具的使用方法。用户可根据输出的性能数据，和异常检测结果，快速定位算子的软、硬件性能瓶颈和异常场景，提升算子性能的分析效率。

## 算子开发样例

|  目录名称                                                   |  功能描述                                              |
| ------------------------------------------------------------ | ---------------------------------------------------- |
| [msProf](./msProf) | 本样例基于Ascend C编程语言实现了MatmulLeakyRelu算子, 同时使用msProf工具调试，给出了对应的端到端实现 |
| [msSanitizer](./msSanitizer) | 本样例通过Ascend C编程语言实现了Add算子，同时展示开发算子过程中，msSanitizer工具可以检测到哪些异常场景提前拦截 |
